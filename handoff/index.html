⚠

If you read nothing else, read this

The agent never has a "run" button. It's always on. The portfolio shows a live indicator (● LIVE · last scan 11s ago) and changes appear via SSE push. The trigger endpoint exists (POST /v1/admin/vendors/:id/scan) but it's admin-only and not surfaced in the product. Production cadence: 6h per vendor. Demo cadence: 60s via SCAN_INTERVAL_SEC env var.
